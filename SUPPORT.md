# Suporte

| Plataforma | Nível |
|---|---|
| Fedora atual com `dnf` | suportada |
| Ubuntu LTS/Debian atual com `apt-get` | suportada |
| WSL 2 com Fedora ou Ubuntu/Debian | suportada |
| Outras distribuições | melhor esforço |

Arquiteturas dependem dos pacotes da distribuição. O instalador requer Python
3.10+, Git, acesso HTTPS e `sudo` para pacotes/shell padrão.
